# Frontend
Application for uploading audio files and viewing transcriptions.

## Prerequisites
- [Node.js](https://nodejs.org/en/download) (v18+ recommended)
- .env file

## Getting started
### Create .env file
```bash 
VITE_BACKEND_SERVICE=http://localhost:8000
```

### Install dependencies
``` bash
npm install
```

### Run development
``` bash
npm run dev
```

### Run tests
``` bash
npm run test
```

### Build image
``` bash
npm run build
```

### Start container
``` bash
docker compose up -d
```
